About EMS

INTRODUCTION OF THE PROJECT

Employee Management system is an application that enables users to create and store Employee Records. The application also provides facilities of a payroll system which enables user to generate Pay slips too. This application is helpful to department of the organization which maintains data of employees related to an organization. It is used to provide the dashreport based on employee registererd in the system. And also keep track the department associated with the employee.

OBJECTIVE OF THE PROJECT

In this world of growing technologies everything has been computerized. With large number of work opportunities the Human workforce has increased. Thus there is a need of a system which can handle the data of such a large number of Employees in an organization. This project simplifies the task of maintain records because of its user friendly nature.
